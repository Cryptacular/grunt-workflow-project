Based on instructions at https://medium.com/@verpixelt/get-started-with-grunt-76d29dc25b01.

Will automatically reload when any html file changes, or if style.scss changes.

Simply download this repo, then follow the instructions on the above page (ie. install Node and Grunt, then npm init on the folder these files are in, and then npm install <dependency> --save-dev for all dependencies listed in gruntfile.js.

When all is done, you can execute grunt and connect to http://0.0.0.0:8000/.

If you want to minify your JS, type grunt uglify.
